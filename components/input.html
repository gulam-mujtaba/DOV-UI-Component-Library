DOV UI Inputs

Input without Validation

Inputs are used for taking some important information from the user so that we can use that information when required.

These type of Inputs are generic in use and used quite often, Eg: Email Address, Numbers

Input with Validation

These type of inputs are used when there are certain conditions to be fullfilled by the user. Eg: Passwords


When the condition is fulfilled, the input section can calidate and turn green.
